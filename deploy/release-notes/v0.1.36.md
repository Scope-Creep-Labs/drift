## v0.1.36

Image-only release.

### What's new

- **Light / dark theme toggle.** A small icon in the sidebar footer cycles through `system → light → dark`. Default is `system` (follows OS `prefers-color-scheme`). Explicit picks persist in `localStorage` and survive reloads. When `system` is selected, an OS dark-mode flip re-themes the SPA live without a reload. Implemented via a Zustand store + MUI `createTheme` variants. (Terminal modal stays dark in both palettes by design, matching the VS Code embedded-terminal convention.)
- **Tool-input parsers accept more shapes.** Smaller models (`gpt-5.4-mini` and friends) reach for Prometheus / Grafana conventions on instinct — `now`, `now-15m`, `1h30m`, bare integers. Drift's parsers used to reject those and force a retry; now they accept everything a model is likely to guess and the schema descriptions bless those forms explicitly so the model doesn't need to learn from a 4xx.

  - `query_range` and `query_logs` (start/end fields) now accept:
    - `'now'` literal
    - `'now-1h'` / `'now+30m'` Grafana-style offsets
    - `'1h'` / `'30m'` / `'7d'` bare durations (interpreted as 'now minus that')
    - ISO 8601 and unix epoch (unchanged)
  - `silence_alert` (`duration` field) now accepts:
    - Single unit: `'45s'`, `'30m'`, `'1h'`, `'7d'`, `'2w'` (week added)
    - Combined: `'1h30m'`, `'2d12h'`, `'1w3d'`, `'90m30s'`
    - Bare integer: `'60'` (interpreted as seconds)

  Error messages on invalid input list every accepted form by name. Net effect: one less wasted tool round-trip per investigation on smaller models, and the larger ones never noticed the constraint anyway.

### Operator action

Image-only — apply via the **Software Updates** modal in the UI (one-click `docker compose pull && up -d`).
